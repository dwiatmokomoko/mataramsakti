## SEO Implementation Strategy

### 1. Location-Based Keywords
```
Primary: "Yamaha [Motor] [Location]"
Secondary: "Dealer Yamaha [Location]"
Tertiary: "Harga [Motor] [Location]"
```

### 2. Content Strategy
- Create location pages for major kecamatan
- Add location mentions in meta descriptions
- Include local landmarks in content
- Add distance/direction from main dealer

### 3. Schema Markup
- LocalBusiness schema for each location
- Service area coverage
- Geo coordinates

### 4. Internal Linking
- Link from main pages to location pages
- Breadcrumb with location
- Related locations sidebar
