## 💻 TECHNICAL IMPLEMENTATION

### Generic Template System
Created a reusable template that works for all motors:

**Benefits:**
- Single source of truth
- Easy to maintain
- Consistent design across all pages
- Dynamic content based on motor data
- Automatic keyword generation

**How it works:**
1. Controller method `motorLandingPage($motorName, $viewName)`
2. Finds motor by name from database
3. Loads all models and variants
4. Passes data to generic view
5. View dynamically generates content

### Code Structure
```php
// HomeController.php
public function nmaxJogja() {
    return $this->motorLandingPage('NMAX', 'nmax-jogja');
}

private function motorLandingPage($motorName, $viewName) {
    $motor = Motor::where('name', 'LIKE', '%' . $motorName . '%')->first();
    $motor->load(['models.variants']);
    $motorModels = $motor->models;
    $relatedMotors = MotorModel::whereHas('motor', function($query) use ($motor) {
        $query->where('category', $motor->category)
              ->where('id', '!=', $motor->id);
    })->take(3)->get();
    
    return view($viewName, compact('motor', 'motorModels', 'relatedMotors'));
}
```
