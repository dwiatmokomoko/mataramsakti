### Tip #4: Keep URLs Clean
```
Good URLs:
✅ /motor/1
✅ /dealer-yamaha-wates
✅ /harga-nmax-jogja

Bad URLs:
❌ /page?id=123
❌ /motor.php?name=nmax
❌ /index.html
```

### Tip #5: Use Canonical URLs
```
Avoid duplicate content:
- http://yamahajogja.id/motor/1
- http://yamahajogja.id/motor/1/
- http://yamahajogja.id/motor/1?ref=fb

Use canonical tag to specify preferred URL
```
